# Base de Datos

Este directorio contiene la estructura de base de datos del proyecto.

## Estructura

- `migrations/` - Migraciones de base de datos
- `seeds/` - Datos de prueba
- `scripts/` - Scripts de utilidad SQL
- `schema/` - Esquemas de base de datos
- `types/` - Tipos TypeScript generados
- `tests/` - Tests de base de datos

## Notas

La base de datos debe configurarse según las necesidades específicas del proyecto.
